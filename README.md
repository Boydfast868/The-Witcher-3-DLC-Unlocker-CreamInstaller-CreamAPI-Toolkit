## 🔍 Troubleshooting Common Issues

Check this list if the tool does not work as expected:

* **The tool does not open:** Ensure you have the latest version of the .NET Framework installed from the Microsoft website.
* **Game does not launch:** Verify that you moved the emulator files into the correct game directory. If you place them in the wrong folder, the game cannot read them.
